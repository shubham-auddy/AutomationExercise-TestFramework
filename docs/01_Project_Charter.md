# Project Charter

## Project Name

AutomationExercise-TestFramework

---

## Project Overview

AutomationExercise-TestFramework is an enterprise-style automation testing framework developed to automate the UI and API functionalities of the Automation Exercise application.

The project is designed to simulate a real-world Software Development Life Cycle (SDLC) and Software Testing Life Cycle (STLC), following industry best practices in planning, framework design, automation development, documentation, version control, continuous integration, and containerized execution.

---

## Objectives

- Build a scalable UI automation framework using Selenium.
- Build a reusable API automation framework using RestAssured.
- Follow clean coding principles and design patterns.
- Produce professional documentation throughout the project.
- Integrate automated execution using GitHub Actions.
- Containerize the framework using Docker.
- Deploy and execute tests using Kubernetes.
- Create an interview-ready and portfolio-quality automation framework.
